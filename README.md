# Nodejs_Mongodb_TodoApp
A simple to do list app built Using Node.js and MongoDB on Cloud9.

For the app to work, you need to install the following packages using **npm install [package name] --save**
- body-parser: 1.18.3
- express: 4.16.4
- mongoose": 5.3.15
